To keep your API key secure, the index.html file in this repository has the API Key field empty by default. This ensures your key is never stolen by bots scanning GitHub.

How to Enable Live AI (For You)

Deploy: Upload index.html and career.html to your GitHub repository and enable GitHub Pages.

Visit: Go to your live website URL (e.g., jaiprakash.github.io).

Unlock:

Scroll down to the AI_NEXUS terminal.

Type the following command and hit Enter:

set_key YOUR_REAL_API_KEY_HERE


The site will save the key to your browser's Secure Local Storage and reload.

The AI will now be LIVE for you on that device!

How it Works for Visitors

Visitors who do not have your key will see the "Simulation Mode".

They can still interact with the terminal.

They will receive pre-written, witty "Demo" responses (e.g., comparing a server crash to cardiac arrest).

This ensures the site always looks broken-free and impressive, even without a live key.
